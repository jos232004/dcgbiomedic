## 🛠️ Tecnologías Usadas

* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Gestión de Datos:** Formato **JSON** (para el almacenamiento y consumo dinámico de datos del sistema)
* **Estilos adicionales:** SCSS / Bootstrap *(borra Bootstrap o SCSS si no los usaste en este proyecto)*

## 📦 Características del Proyecto

* **Diseño Responsivo:** Adaptado para que se vea perfecto en computadoras, tablets y celulares.
* **Datos Dinámicos con JSON:** Los datos de ciertos módulos (como mantenimientos o servicios) se cargan de forma dinámica desde archivos estructurados en formato JSON, mejorando el mantenimiento del sitio.
